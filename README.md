###### โครงการม็อดไม่แสวงหาผลกำไรขอขอบคุณ
+ ศรีราชา และ Noto Sans Thai โดย [คัดสรรดีมาก](https://www.cadsondemak.com/)
+ Noto Sans โดย [กูเกิล](https://fonts.google.com/noto)
